# 📊 TQPA Calculator Chrome Extension

A lightweight Chrome extension that automates the calculation of a student’s **Total Quality Point Average (TQPA)** directly from their academic portal.  
This project was built as a practical application of systems thinking, problem decomposition, and rapid prototyping in Computer Engineering.

---

## 🚀 Features
- Extracts grades and units from the student portal table.
- Calculates **weighted TQPA** automatically.
- Displays results in a clean popup interface.
- Reduces manual computation errors and saves time.
